About Microsoft Users Group Hyderabad

A few years ago, an idea was born it was and is still popularly known as DNUG (DotNET Users Group) Hyderabad, which today has matured and stands in front of you transformed as Microsoft Users Group Hyderabad aka MUGH.

Although MUGH started life as DNUG (.NET Users Group), we felt that various Microsoft technologies function so well in concert with each other and our earlier name did restrict our scope to just one of them. Today we have special groups branched into various fields under MUGH, namely DNUG (a dot net user group), IT Pro (a user group for it community), SQLCon (India's first SQL community), a student’s group and supports various other user groups from different regions.

MUGH is operated by a core group that consists of Microsoft MVPs and Community Stars. Microsoft often helps us in conducting our activities that include user group meetings, developer conferences, annual events.

Presently, MUGH's user group meetings are held once every month, you need to be a MUGH member to attend these meets.
